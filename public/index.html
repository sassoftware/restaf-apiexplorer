Important Usage Note

  • Ask your system administrator to enable CORS using the SAS Environment Manager. If you own the server you should logon as administrator to Environment Manager and do the following:
    • configuration ->(Definitions from the view dropdown)
    • Select sas.commons.web.security. Make sure allowedHeaders, allowedMethods and AllowedOrigin are set to * or some whitelist
  • Ask your system administrator for a clientid and clientsecret. Depending on your company policies this will be either for password flow or implicit flow
